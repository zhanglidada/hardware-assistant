# 安全的数据更新指南

## ⚠️ 为什么不推荐使用爬虫

1. **法律风险**: 违反网站服务条款
2. **技术风险**: 
   - Cloudflare 等反爬虫保护很强
   - IP 可能被封禁
   - 网站结构变化导致爬虫失效
3. **维护成本**: 需要持续维护和调试
4. **不稳定**: 随时可能失败

## ✅ 推荐的安全方案

### 方案 A：使用现有数据（最推荐）

您当前已有：
- **CPU**: 100 条高质量数据（AMD + Intel）
- **GPU**: 25 条数据（NVIDIA + AMD）  
- **Phone**: 25 条数据（多品牌）

**这些数据已经足够支持一个硬件参数小助手项目！**

### 方案 B：手动扩展数据

如果需要更多数据，推荐手动收集：

1. **访问官方网站**：
   - Intel ARK: https://ark.intel.com/
   - AMD 规格页面: https://www.amd.com/zh-hans/processors
   - NVIDIA 显卡库: https://www.nvidia.com/zh-cn/geforce/graphics-cards/
   
2. **使用数据模板**：
   ```python
   # 复制 skills/scripts/scrapers/cpu.py 中的数据格式
   {
       "id": "cpu-xxx",
       "model": "Intel Core i7-14700K",
       "brand": "Intel",
       "releaseDate": "2023-10-17",
       "price": 3199,
       "cores": "8P+12E",
       "threads": "28",
       "baseClock": 3.4,
       "boostClock": 5.6,
       "socket": "LGA1700",
       "tdp": 125,
       "cache": 33,
       # ... 其他字段
   }
   ```

3. **手动添加到对应的数组中**

### 方案 C：使用公开 API（如果可用）

查找是否有官方或第三方硬件数据 API：
- CPU-World API
- PC Part Picker（需要申请）
- 其他硬件数据库

### 方案 D：定期手动更新

建议每季度手动补充 5-10 条新产品数据，而不是依赖不稳定的爬虫。

## 📋 手动更新流程

1. 选择要添加的产品（通常是新发布的旗舰产品）
2. 从官方网站获取准确规格
3. 按照现有格式添加到 JSON 数组中
4. 运行验证：`python3 skills/scripts/update_db.py`
5. 检查数据完整性

## 🛡️ 如果坚持使用爬虫的安全建议

**不推荐，但如果必须使用：**

1. **降低频率**: 
   - 请求间隔 5-10 秒
   - 每天最多爬取 50 条数据
   
2. **使用代理**:
   - 使用代理池轮换 IP
   - 避免使用免费代理
   
3. **尊重 robots.txt**:
   - 检查网站的爬虫规则
   - 遵守 Crawl-delay
   
4. **错误处理**:
   - 遇到 403/429 立即停止
   - 不要无限重试
   
5. **备用方案**:
   - 爬虫失败时使用静态数据
   - 不要让项目依赖爬虫

## 🎯 我的建议

**使用方案 A + 方案 B 的组合**：
- 保持现有的 100 条 CPU 数据
- 每季度手动添加 5-10 条新产品
- 完全可控、稳定、合法

这样您的项目：
✅ 稳定可靠
✅ 数据质量高
✅ 无法律风险
✅ 易于维护
✅ 不依赖外部服务

---

**记住**: 一个成功的项目靠的是**稳定可靠的数据**，而不是数据的数量。
